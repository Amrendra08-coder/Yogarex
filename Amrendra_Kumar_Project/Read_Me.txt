YogaRex 🧘‍♂️
YogaRex is a personal web development project built with HTML5, CSS3, and JavaScript. It’s a non-live, interactive yoga guide designed to enhance my front-end development skills.

Features ✨
Step-by-step yoga pose guidance

Responsive and user-friendly design

Built-in timer for practice sessions

Smooth animations and transitions

Technologies Used 🛠️
HTML5

CSS3 

JavaScript 

How to Use 🚀
Open login.html in your browser.

To access the main page (main_index.html) there are two ways: either directly by clicking on main_index.html or by visiting the login page and clicking the "Login Now" button in the form area.

Purpose 🎯
This project is for educational purposes only, aimed at improving my understanding of front-end development. It’s not intended for production use.
